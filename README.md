# 🧠 Python Quick Revision Notes

## 🟢 Basics

### Variables
- Used to store data

 python
x = 10
name = "Mukesh"
 `

### Data Types

* int, float, str, list, tuple, dict, set

 python
a = 5        # int
b = 5.5      # float
c = "hello"  # string
 

### Input & Output

 python
name = input("Enter name: ")
print(name)
 

***

## 🔵 Operators

* Arithmetic: +, -, \*, /, %
* Comparison: ==, !=, >, <
* Logical: and, or, not

 python
print(5 > 3)   # True
 

***

## 🟡 Conditions

 python
x = 10

if x > 5:
    print("Greater")
elif x == 5:
    print("Equal")
else:
    print("Smaller")
 

***

## 🔁 Loops

### For Loop

 python
for i in range(5):
    print(i)
 

### While Loop

 python
i = 0
while i < 5:
    print(i)
    i += 1
 

***

## ⚙️ Functions

* Reusable block of code

 python
def add(a, b):
    return a + b

print(add(2, 3))
 

***

## 📦 Data Structures

### List

 python
nums = [1, 2, 3]
nums.append(4)
 

### Tuple

 python
t = (1, 2, 3)
 

### Dictionary

 python
d = {"name": "Mukesh", "age": 25}
print(d["name"])
 

### Set

 python
s = {1, 2, 3}
 

***

## 🧱 OOP (Object-Oriented Programming)

### Class & Object

 python
class Person:
    def __init__(self, name):
        self.name = name

p1 = Person("Mukesh")
print(p1.name)
 

### Inheritance

 python
class Student(Person):
    pass
 

***

## 📂 File Handling

 python
# Write
with open("file.txt", "w") as f:
    f.write("Hello")

# Read
with open("file.txt", "r") as f:
    print(f.read())
 

***

 python
